# AzeriteUI — JuNNeZ Edition

**This is an unofficial fan-edited version of [AzeriteUI 5](https://github.com/goldpawsstuff/AzeriteUI5) by GoldpawsStuff.**

This fan edition contains personal modifications, bug fixes, and experimental features not present in the official release. For the official, fully-supported version of AzeriteUI, please visit the original author's projects.

---

## Description

AzeriteUI is a custom World of Warcraft user interface designed for modern WoW (Retail). This JuNNeZ Edition includes:

- Additional bug fixes for WoW 12+ (Midnight expansion)
- Custom tweaks and quality-of-life improvements
- Experimental features and refinements

**Note:** This is a personal project maintained independently. For official support, use the original AzeriteUI or visit the Discord where I have a channel for this addon.

## FAQ

**Q: How can I remove abilities from the action bars?**  
A: Hold down `Alt+Ctrl+Shift` and drag with the left mouse button.

**Q: Is this compatible with the official AzeriteUI?**  
A: No. Do not install both versions simultaneously. This fan edition replaces the official addon.

**Q: Where can I get support?**  
A: This is a personal fan project. For official AzeriteUI support, join GoldpawsStuff's Discord (see below).

## Configuring the UI
- `/azerite` — Toggles the options menu
- `/lock` — Toggles movable frame anchors (use EditMode for default Blizzard frames)
- `/resetsettings` — Full settings reset (emergency use only — erases all profiles)
- `/azdebug` — Opens debug menu (development mode must be enabled in options)

## GitHub Cloud Dev Environment

This repository now includes a GitHub Codespaces dev container:
- `.devcontainer/devcontainer.json`
- `.devcontainer/Dockerfile`

### Start in Codespaces
1. Open the repository on GitHub.
2. Click `Code` -> `Codespaces` -> `Create codespace on main`.
3. Wait for container build to finish.

The container installs:
- Lua 5.1 (`lua`, `luac`)
- `luarocks`
- `ripgrep`
- Recommended WoW/Lua VS Code extensions used in this project
