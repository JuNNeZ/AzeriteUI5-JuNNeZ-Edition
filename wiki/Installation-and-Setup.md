# Installation and Setup

## Install
1. Download from:
   - GitHub Releases: https://github.com/JuNNeZ/AzeriteUI5-JuNNeZ-Edition/releases/latest
   - CurseForge: https://www.curseforge.com/wow/addons/azeriteui-junnez-edition-wow12
2. Extract folder AzeriteUI5_JuNNeZ_Edition into:
   World of Warcraft/_retail_/Interface/AddOns/
3. Restart the game or run /reload.

## First run
- Open options with /az
- Move AzeriteUI frames with /lock
- Use Blizzard Edit Mode for Blizzard-owned frames

## Important
- Do not install this alongside official AzeriteUI.
- This addon targets WoW Retail only.

## Validate install
- Addon appears in AddOns list at character select.
- AzeriteUI frames are visible after login.
- /az opens the settings panel.
