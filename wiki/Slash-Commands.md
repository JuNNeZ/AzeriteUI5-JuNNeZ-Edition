# Slash Commands

## Player commands
- /az
- /azerite
  - Opens AzeriteUI options.
- /lock
  - Toggles AzeriteUI frame movement mode.
- /resetsettings
  - Full settings reset for AzeriteUI (all profiles).
- /saiyaratt
  - Applies the SaiyaRatt preset profile/layout path.
- /setminimaptheme <name>
  - Applies a minimap theme from the supported list.

## Debug command
- /azdebug
  - Opens debug menu (development mode required).

## Common /azdebug subcommands
- /azdebug health on|off|toggle
- /azdebug healthchat on|off|toggle
- /azdebug bars on|off|toggle
- /azdebug fixes on|off|toggle
- /azdebug dump target|player|tot|all
- /azdebug aurasnapshot player|topright|both
- /azdebug nameplates [unit]
- /azdebug snapshot [unit]
- /azdebug blizzard enable
- /azdebug scale
- /azdebug scale nameplates [unit]
- /azdebug scale reset
- /azdebug keys <subcommand>
- /azdebug raidbar status
- /azdebug raidbar on|off|toggle
- /azdebug scripterrors
- /azdebug secrettest [unit]

## Notes
- Use /reload after major setting changes.
- /resetsettings is destructive.
