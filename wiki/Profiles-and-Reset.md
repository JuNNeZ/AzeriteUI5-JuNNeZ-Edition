# Profiles and Reset

## Profile operations
From /az -> Settings Profile you can:
- Select active profile
- Create new profile
- Duplicate a profile
- Delete a non-default profile
- Reset current profile

## Full reset command
- /resetsettings
  - Resets AzeriteUI saved database (all profiles)
  - Use only for hard recovery

## Recommended backup workflow
1. Create a named profile before large experiments.
2. Duplicate profile before risky layout changes.
3. Use profile reset before full database reset.
