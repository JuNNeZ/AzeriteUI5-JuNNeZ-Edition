# Action Bars

AzeriteUI supports up to 8 main bars plus pet and stance bars.

## Global behavior
- Hide hotkeys
- Cast on key down
- Command binding behavior
- Inactive dimming controls

## Per-bar controls
- Enable/disable
- Number of buttons
- Layout mode
- Growth direction
- Padding and line breaks
- Fade behavior and mouseover rules

## Pet and stance bars
Pet and stance bars expose equivalent layout/fade controls where applicable.

## Practical tips
- Start with fewer active bars and expand as needed.
- Use /lock to verify anchors while tuning layouts.
- Use /reload after large structural changes.
