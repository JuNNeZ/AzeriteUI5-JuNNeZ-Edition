# AzeriteUI — JuNNeZ Edition Wiki

Welcome to the official documentation wiki for **AzeriteUI — JuNNeZ Edition**, a complete World of Warcraft user interface replacement for Retail (WoW 12 — Midnight expansion).

> **This is an unofficial fan-edited version of [AzeriteUI 5](https://github.com/goldpawsstuff/AzeriteUI5) by GoldpawsStuff**, maintained and extended by JuNNeZ with WoW 12 compatibility fixes, bug patches, and quality-of-life improvements.

---

## What is AzeriteUI?

AzeriteUI replaces the entire default Blizzard user interface with a clean, art-driven design inspired by the Azerite aesthetic. It provides:

- **Custom unit frames** for every unit type (player, target, focus, party, raid, boss, arena)
- **Up to 8 configurable action bars** plus pet and stance bars
- **Custom nameplates** with smart interrupt coloring
- **Explorer Mode** — an immersive UI fade system
- **Custom aura displays** with advanced filtering
- **Minimap, chat, tooltips, world map** overrides
- **A single unified options menu** via `/az`

No initial configuration is required — AzeriteUI works out of the box with sensible defaults.
