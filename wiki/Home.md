## Supported version
- WoW Retail (Midnight / WoW 12+)

## Main commands
- /az or /azerite : Open options
- /lock : Toggle AzeriteUI frame movers
- /resetsettings : Reset addon database (all profiles)
- /setminimaptheme <name> : Change minimap theme
- /azdebug : Open debug tools (development mode required)

## About this edition
This is an unofficial fan-maintained edition of AzeriteUI 5 by GoldpawsStuff, with ongoing retail compatibility fixes and QoL updates.
