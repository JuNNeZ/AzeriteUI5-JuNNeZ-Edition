## Typical setup flow
1. Unit Frame Settings: enable/disable frame groups and tune aura/castbar behavior.
2. Action Bars: set bar count, layout, and fading.
3. Nameplates: adjust scale, health text placement, and castbar color behavior.
4. Explorer Mode: tune fade behavior and exit conditions.
5. Frame Movement: move AzeriteUI frames with /lock then fine tune here.

## High-impact options
- Prioritize Unit Frame Auras
- Show Blizzard Raid Bar
- Nameplate health text placement
- Bar fading and dimming behavior
- Explorer Mode exit conditions

## Profile location
Use Settings Profile to create, switch, duplicate, delete, and reset profiles.
