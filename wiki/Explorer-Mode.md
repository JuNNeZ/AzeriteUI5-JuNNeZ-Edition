# Explorer Mode

Explorer Mode manages automatic UI fading behavior for immersive gameplay.

## What it controls
- Fade timing after login/reload/loading/combat
- Conditions that force UI visible
- Which UI areas should fade

## Common exit conditions
- Enter combat
- Low health/power thresholds
- Group/instance state
- Targeting and vehicle states

## Suggested setup
1. Enable Explorer Mode in /az.
2. Keep conservative delays first.
3. Test in open world, dungeons, and raids.
4. Tune until visibility vs immersion feels right.
