# FAQ

## Is this the official AzeriteUI?
No. This is the JuNNeZ fan-maintained edition based on AzeriteUI 5.

## Which game version is supported?
WoW Retail (WoW 12 / Midnight+).

## How do I open settings?
Use /az or /azerite.

## How do I move frames?
Use /lock for AzeriteUI-owned frames. Use Blizzard Edit Mode for Blizzard-owned frames.

## How do I reset everything?
Use /resetsettings. This is destructive and resets all AzeriteUI profiles.

## Does it support profiles?
Yes. Use /az -> Settings Profile for create/switch/duplicate/delete/reset.

## Why do some values behave differently in WoW 12?
WoW 12 introduced secret-value restrictions for some combat data paths. AzeriteUI includes compatibility handling so UI remains stable.

## What if nameplate interrupt colors look wrong?
Check Nameplates options first, then test without third-party nameplate addons, then capture /azdebug nameplates output for a sample cast.

## Where do I report problems?
GitHub Issues: https://github.com/JuNNeZ/AzeriteUI5-JuNNeZ-Edition/issues
