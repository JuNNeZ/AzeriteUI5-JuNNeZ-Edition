# Unit Frames

AzeriteUI includes per-unit modules with independent toggles and options.

## Core frame groups
- Player
- Player Alternate
- Target
- Target of Target
- Focus
- Pet
- Boss
- Arena
- Party
- Raid 5
- Raid 25
- Raid 40
- Player Castbar
- Player Class Power

## Shared controls
- Aura prioritization/ordering
- Castbar text color by interrupt state
- Blizzard raid utility bar visibility

## Party and raid highlights
- Visibility rules by group size/content type
- Health color modes
- Range fading
- Debuff emphasis behavior

## Player frame highlights
- Custom artwork and resource presentation
- Castbar integration
- Aura row behavior
- PvP/indicator presentation options

## Target frame highlights
- Aura layout controls
- Castbar and interrupt readability
- Health/power text presentation controls
